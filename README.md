# Customer Segmentation Analysis

This project analyzes customer purchase behavior using MySQL and visualizes the results in a spreadsheet.

Key Insights:
- Total spent per customer
- Average order value
- Purchase frequency
- High, medium, low-value customer segmentation

Tools:
- MySQL
- Google Sheets / Excel
- SQL queries: SUM, AVG, COUNT, GROUP BY, JOINs

Project demonstrates ability to segment customers and generate actionable business insights.
